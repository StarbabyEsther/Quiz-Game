# Java Quiz Game 

A multiple choice Java quiz game with 5 questions and a scoring system, built in Java 

## About
A 5 question multiple choice quiz about Java 
programming. The player picks from 5 options 
(A-E) for each question and gets a final 
score at the end.

## What I learned
- How to use 2D arrays String[][] to store 
  multiple options for each question
- How nested for loops work — a loop inside 
  a loop to go through each question and 
  its options
- How to store single letters as char and 
  compare them to check answers

## Built with
- Java
